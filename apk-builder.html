KisanVaani APK Builder Instructions

This guide walks you through generating an Android APK for KisanVaani using free tools since the web app is now fully PWA-ready.

Section 1 — METHOD A: AppsGeyser (5 minutes, no coding)

  1. Go to appsgeyser.com
  2. Click "Create App" → choose "Website"
  3. URL: https://yourusername.github.io/kisanvaani
  4. App Name: KisanVaani
  5. Upload icon (green + wheat)
  6. Click Create → Download APK
Open AppsGeyser →

Section 2 — METHOD B: WebIntoApp (better quality)

  1. Go to webintoapp.com
  2. Enter URL: your Vercel or GitHub Pages URL
  3. Set App name, icon, package name
  4. Build → Download APK (free tier)
Open WebIntoApp →

Section 3 — METHOD C: GoNative (most professional)

  1. Go to gonative.io
  2. Enter your Web App URL
  3. Customize the icon, UI colors
  4. Download trial APK (free)
Open GoNative →

Section 4 — After getting APK

  1. Test the APK on your own Android phone first.
  2. Upload APK to GitHub Releases (github.com → your repo → Releases → New Release → Attach APK file → Publish).
  3. Update APK download link in index.html so the homepage properly routes to your KisanVaani.apk.
  4. Share via WhatsApp to farmers!
